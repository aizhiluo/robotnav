# Auto Drive

This repository contains a collection of software for research on autonomous vehicles.

## Repository structure

| Folder      | Description              |
| ----------- | ------------------------ |
| cmake       | cmake configuration      |
| data        | experiment results, logs |
| docs        | ocumentation             |
| maps        | map files                |
| src         | C++ code                 |

## Build and run

### Requirements

* C++17

### Install dependencies

### Compile code

```
$ mkdir build
$ cmake ..
$ make -j8
```
